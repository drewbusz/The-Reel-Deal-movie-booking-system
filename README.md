# Reel Deal Movie Booking System

Drew Busz  
Midterm Project

## Overview
Reel Deal is a movie booking system designed to make theater reservations simple and reliable. The current midterm prototype demonstrates core booking functionality using Python and MySQL.

## What is Included
- 1_code: Python scripts, database schema, and README1.txt
- 5_documentation: system requirements document, brochure, and presentation slides

## Current Midterm Features
- User login with role-based access
- Seat availability display for a showtime
- Seat reservation through a database-backed workflow
- Booking record storage in MySQL
- Seat status updates to help prevent double booking

## Future Development
Future development will include movie and showtime selection, manager functions for maintaining movies and showtimes, expanded booking features, and a more complete user interface.
