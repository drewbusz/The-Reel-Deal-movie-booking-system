The Reel Deal Movie Booking System

Test Case 1: Customer Login
Steps:
1. Open http://127.0.0.1:5000.
2. Log in using the customer test account.
3. Confirm that the customer is redirected to the Movies page.

Expected Result:
The customer successfully logs in and sees the Movies page with movie posters, descriptions, and showtime buttons.

Status:
Passed


Test Case 2: Movie and Showtime Display
Steps:
1. Log in as the customer.
2. View the Movies and Showtimes page.
3. Confirm that movies, descriptions, posters, and showtime buttons are displayed.

Expected Result:
The Movies page displays movie data pulled from the MySQL database.

Status:
Passed


Test Case 3: Multi-Seat Booking
Steps:

Passed
